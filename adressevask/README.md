Adressevasekeren tager CSV med adresser og geokoder dem ved brug af [DAWA's datavask API](http://dawa.aws.dk/adressedok#datavask).
[PapaParse](http://papaparse.com/) bruges til at parse CSV til JSON og efter geokodningen dannes [GeoJSON](https://www.npmjs.com/package/geojson).

[MapBox](https://www.mapbox.com/mapbox.js/api/v2.4.0/) er brugt til kortet
